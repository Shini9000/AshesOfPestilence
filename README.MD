# Ashes of Pestilence! V0.3 GUIDE
Executable JAR Text adventure game!

#### Character guidelines:
When creating a character *new game* you are greeted with multiple options.<br>
These options are very important for stats.

Races: Human, Orc, Elf and Dwarf  
Classes: Hunter, Mage, Warrior and Assassin
